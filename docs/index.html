Simple Tooltips Actually not exactly a tooltips

How to create a simple tooltip like from the right of this text?

  1. Add style file dist/touch-tooltip.css inside your <head> tag; 
    <link href="dist/touch-tooltip.css" rel="stylesheet"/>
  2. Add some element with any you've liked content and with class Content somewhere on your page; 
    <div class="Content">
    [tooltip's content]
</div>
  3. Wrap your Tooltip's content element with another element with classes TouchTooltip There is the base class of tooltip Bottom Show tiny arrow at the bottom of tooltip Default Apply the Default color theme ;
  4. Would be better if you'll add dist/touch-tooltip.js right after jQuery. 
    <script src="[path-to-jQuery]/jquery.min.js"></script>
<script src="dist/touch-tooltip.js"></script>
[tooltip's content] 
<div class="TouchTooltip Bottom Default">
    <div class="Content">
        [tooltip's content]
    </div>
</div>

Tooltips in text

Simplest way to use the TouchTooltip with text is two steps only:

  1. Wrap your target word with inline element <span> for example with class ApplyTooltip <span class="ApplyTooltip">inline element</span> ;
  2. Put inline TouchTooltip All elements inside must be inline.
    Like <span>     right after <span class="ApplyTooltip"> right after <span class="TouchTooltip Top Left Default"> <span class="Content"> Tooltip's content </span> </span> </span> your target word;

</span>

Understanding of centering

While horizontal position Left or Right of arrow is not defined, Bottom/Top means "show an arrow at center bottom/top of tooltip".

Arrow Bottom and center 
<div class="TouchTooltip Bottom Default">
    <div class="Content">
        Arrow Bottom and center
    </div>
</div>

But when you define any size Small, Medium, Large, Huge it means: "show an arrow at center bottom/top of tooltip and put whole tooltip to center of his parent or word."

Arrow Bottom and center
Centered 
<div class="TouchTooltip Medium Bottom Default">
    <div class="Content">
        Arrow Bottom and center<br/>
        Centered
    </div>
</div>

Customizing values

Find config constants and change them as you want. That's all. So easy.
I'm almost sure you will deal with it and there is no need to explain what each constant means. 

$SMALL_SIZE: 150px;
$MEDIUM_SIZE: 250px;
$LARGE_SIZE: 350px;
$HUGE_SIZE: 450px;

$INNER_PADDING: 10px;

$SHADOW_OFFSET_LEFT: 0;
$SHADOW_OFFSET_TOP: 2px;
$SHADOW_STRENGTH: 4px;

$BORDER_RADIUS: 5px;
$BORDER_WIDTH: 1px;

$ARROW_SIZE: 6px;

Creating a theme

Scroll to the end of file or use text search for YourNewTheme. That theme is gradient style, so you need only three colors:

  1. Top color of gradient;
  2. Bottom color of gradient;
  3. Text color (usually black or white).
&.YourNewTheme {
    $top-color: #eef3f5;
    $bottom-color: #d8e3e9;
    $text-color: #000;
    @include defaultTheme($top-color, $top-color, $text-color);
}

JavaScript

The first and most important — you really need jQuery to execute JavaScript part of TouchTooltip.

Secondly — if an user turn JavaScript off — most of features will still work. But, unfortunately, not all. Here is a list of JavaScript-depended features:

  • Quick ApplyTooltip;
  • Left Center and Right Center positions during inline usage (height is not defined)
